Bring back thorn/þorn to the web with this Firefox extension.

== FEATURES

* **ORTHOGRAPHIC RESTORATION** {em} Automatically swaps modern `th` digraphs for the majestic `þ`.
* **SMART CASE MAPPING** {em} Respects capitalisation rules to ensure `the`, `The` and `THE` become `þe`, `Þe` and `ÞE` respectively. Similar applies for Eth.
* **OPTIONAL ETH SUPPORT** {em} Toggleable `dh` to `ð` conversion. When disabled, `dh` is treated as a variant of thorn.
* **LIVE MUTATION** {em} Uses `MutationObserver` to catch and convert new text in real-time on infinite-scroll sites like Reddit or X.
* **BLACKLIST** {em} Features a built-in blacklist to exclude specific domains.
* **DEVELOPER FRIENDLY** {em} Skips sensitive HTML tags like `<code>`, `<pre>` and `<input>` to prevent breaking technical content or form submissions.
